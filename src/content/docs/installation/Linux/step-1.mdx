To install SplashKit, you will firstly need to install some command line tools using the apt or apt-get package manager.

## Steps

1. Open up a terminal.

    ![Opening a terminal in Linux](/gifs/linux/open-terminal.gif)

2. **Update your system** to make sure all packages are up-to-date before installing new tools.

    <Tabs syncKey="linux-distro">
    <TabItem label="Ubuntu/Debian">
    ```bash
    sudo apt-get update && sudo apt-get upgrade
    ```
    </TabItem>

    <TabItem label="Arch (pacman)">
    ```bash
    sudo pacman -Syu
    ```
    </TabItem>

    <TabItem label="Arch (yay)">
    ```bash
    yay -Syu
    ```
    </TabItem>

    <TabItem label="Fedora">
    ```bash
    sudo dnf update
    ```
    </TabItem>

    <TabItem label="Alpine">
    ```bash
    sudo apk update && sudo apk upgrade
    ```
    </TabItem>
    </Tabs>

3. Install Command Line Tools: **curl** and **git**, which will be used in the SplashKit install process.

    Install using your package manager. For example:

    <Tabs syncKey="linux-distro">
    <TabItem label="Ubuntu/Debian">
    ```bash
    sudo apt-get install curl git
    ```
    </TabItem>

    <TabItem label="Arch (pacman)">
    ```bash
    sudo pacman -S curl git
    ```
    </TabItem>

    <TabItem label="Arch (yay)">
    ```bash
    yay -S curl git
    ```
    </TabItem>

    <TabItem label="Fedora">
    ```bash
    sudo dnf install curl git
    ```
    </TabItem>

    <TabItem label="Alpine">
    ```bash
    sudo apk add curl git
    ```
    </TabItem>
    </Tabs>
